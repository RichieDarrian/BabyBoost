haloo, berikut cara run website-nya hanya dalam 5 langkah yang mudah! 😲
1. Pastikan Python sudah terinstalasi. Jika belum, silakan download dan install dari "https://www.python.org/downloads/". Saat proses instalasi, jangan lupa centang opsi "Add Python to PATH".
2. Jika Anda user Windows, klik kanan file "run-windows.bat" di folder BabyBoost. Jika Anda user Mac OS, klik kanan file "run-mac.sh". Pilih "Run as administrator".
3. Bila muncul window "Windows protected your PC", klik "More info" lalu "Run anyway".
4. Selamat mencoba website kami! ^v^
5. Untuk mematikan website, tekan ctrl + c dalam terminal.

Sangat mudah kan? 😲
